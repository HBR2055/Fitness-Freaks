# Fitness-Freaks# 🏋️ FitnessFreaks – AI Powered Fitness Platform

FitnessFreaks is a modern AI-powered fitness web application developed as a Final Year Project (FYP). The platform provides personalized workout plans, nutrition guidance, progress tracking, BMI analysis, community interaction, and an AI fitness assistant to help users achieve their health and fitness goals.

## 📌 Features

### 👤 User Authentication
- Secure user registration and login
- User profile management
- Personalized dashboard

### 🤖 AI Fitness Coach
- AI-powered workout recommendations
- Personalized fitness guidance
- Exercise suggestions based on user goals

### 💪 Workout Management
- Browse workout plans
- Save favorite workouts
- Workout categorization
- Exercise details and instructions

### 🥗 Nutrition Plans
- Personalized nutrition recommendations
- Meal planning
- Nutrition tracking
- Healthy diet suggestions

### 📊 Progress Tracking
- Daily progress logging
- Weight tracking
- Workout history
- Fitness reports
- Performance analytics

### ⚖️ BMI Calculator

- BMI classification
- Health recommendations

### 👥 Community

- Fitness discussions

### 📝 Blog System

- Nutrition advice

### 👨‍💼 Admin Panel

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/your-username/FitnessFreaks.git
```

Navigate to the project directory

```bash
cd FitnessFreaks
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
